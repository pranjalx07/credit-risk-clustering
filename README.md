# Credit Risk Clustering

An unsupervised ML project using KMeans clustering on German Credit Data.

## Libraries Used
- numpy, pandas, matplotlib, seaborn
- scikit-learn (KMeans, silhouette score)
- scipy (Box-Cox transformation)

## How to Run
1. Install dependencies: `pip install numpy pandas matplotlib seaborn scikit-learn scipy`
2. Open `main.ipynb` in Jupyter Notebook
